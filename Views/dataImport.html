Make Dashboard

by Qlik

Posted on February 5, 2018 at 12:00 PM

1. Clik on “Create new app” button to create a new dashboard sheet.

2. Click on “Open app” and choose the option “Add data from files and other sources” to import data from REST API or Excel

3. To import REST API data click on REST button. Key in the URL, method, authentication schema and query headers (if exist) and click on “create” button. Click here for guidelines to import REST API data

4. Click on “Add data” after necessary changes made to the data. Click on “table” button to edit the data.

5. Click on the edit icon in the table “rsrp” to change the name of the field, to create a new field and so on.

6. Click on “Load data” after making necessary changes. Click on edit the sheet upon receiving a pop up menu.

7. Click on charts and select “line chart” to project a line chart into the sheets.

8. Click on the add dimension and select Time to add it as the X-axis value.

9. Click on Add measure and select rsrp followed by Sum(rsrp) to add it as the y-axis value.

10. Click on Appearance and select X-axis and change the labels and title to “title only”.

11. Click on the title to edit the title of the line chart.

12. Click on “done” when done with editing.

13. The header of the page can be edited by clicking on the “…” next to the title of the sheet.

14. To create a webpage with using the sheets created, go to home and click on “…” located at the most right of the page and select “Dev Hub”.

15. Click on “mashup editor” to make a webpage with the sheets data.

16. Click on create new mashup to create a new page template. Select Grid mashup template for the page.

17. Click on the “select an app” to select the sheets to be included into the page.

17. Drag and drop the charts on the left panel and drop it on the desired layout in the page.

18. Html page can be edited by accessing the html tab.
