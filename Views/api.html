URL:Get 1 day Time Series data

https://oip.tmrnd.com.my:8083/api/plugins/telemetry/DEVICE/ID/values/timeseries?keys=rsrp&startTs=1517356800000&endTs=1517452579609&interval=60000&limit=100&agg=NONE

Parameters

keys=rsrp
startTs=1517356800000
endTs=1517452579609
interval=60000
limit=100
agg=NONE

How to get the DEVICE ID?

Open Thingsboard, go to devices and click on “copy device id” button to copy the ID. Replace the ID with copied values from clipboard.
